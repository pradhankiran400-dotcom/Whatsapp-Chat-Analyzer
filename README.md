<h1>💬 WhatsApp Chat Analyzer</h1>
<h2>Analyze and visualize your WhatsApp chat exports — discover messaging patterns, emoji usage, activity trends, and more.</h2>

<h3>✨ Features</h3>
<h4>
📊 Message & word count per user <br>
📅 Daily, monthly & hourly activity Bar chart <br>
😂 Emoji analysis <br>
☁️ Word cloud <br>
🔗 Media & link tracking
</h4>

<h3>📤 Export Your Chat</h3>
<h4>
Open WhatsApp → Chat → ⋮ Menu → More → Export Chat <br>
Choose Without Media <br>
Upload the .txt file to the app
</h4>

<h3>🛠 Tech Stack <br>
Python · Streamlit · Pandas · Matplotlib · WordCloud </h3>
