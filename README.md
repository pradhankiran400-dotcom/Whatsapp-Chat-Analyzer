💬 WhatsApp Chat Analyzer
Analyze and visualize your WhatsApp chat exports — discover messaging patterns, emoji usage, activity trends, and more.

✨ Features

📊 Message & word count per user
📅 Daily, monthly & hourly activity Bar chart
😂 Emoji analysis
☁️ Word cloud
🔗 Media & link tracking

📤 Export Your Chat

Open WhatsApp → Chat → ⋮ Menu → More → Export Chat
Choose Without Media
Upload the .txt file to the app

🛠 Tech Stack
Python · Streamlit · Pandas · Matplotlib · WordCloud
